# 🔍 Digital Forensics Lab

This repository contains all experiments performed as part of the **Digital Forensics Lab**.
It covers various techniques used in forensic investigation such as disk imaging, file recovery, memory analysis, metadata extraction, and browser/email forensics.

---

## 📁 Experiments Included

### 🧪 Exp1: Disk Imaging

* Created disk images for forensic analysis
* Used tools to ensure data integrity
* Script: `disk_imaging.sh`

### 🧪 Exp2: File Recovery

* Recovered deleted/lost files
* Analyzed storage for hidden data
* Script: `recover_files.py`

### 🧪 Exp3: Memory Analysis

* Analyzed RAM dumps for evidence
* Identified running processes and suspicious activity
* Script: `memory_analysis.py`

### 🧪 Exp4: Timeline Analysis

* Generated activity timeline from system data
* Correlated events for investigation
* Script: `generate_timeline.py`

### 🧪 Exp5: Metadata Analysis

* Extracted metadata from files
* Identified hidden information
* Script: `extract_metadata.py`

### 🧪 Exp6: Email Header Analysis (PGP)

* Analyzed email headers
* Verified authenticity and routing
* Script: `email_forensics.py`

### 🧪 Exp7: Browser History Analysis

* Investigated browsing activity
* Extracted user behavior patterns
* Script: `browser_forensics.py`

### 🧪 Exp8: Android Image Analysis

* Performed mobile device forensics
* Analyzed Android image data
* Script: `android_forensics.py`

---

## 🛠️ Tools & Technologies Used

* Autopsy
* FTK Imager
* Wireshark
* Python
* Linux Commands

## 🎯 Objective

To understand and apply digital forensic techniques for identifying, collecting, analyzing, and preserving digital evidence.
